## Node
Download Node Version Manager (NVM) to download and manage Node versions.
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

Once that installs, run the following command:
```
nvm install stable
```
